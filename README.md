**I'm a passionate software engineering undergraduate** at **NSBM Green University**, driven by the intersection of clean code, compelling UI, and scalable systems architecture.

My work spans **cross-platform mobile development** with Flutter & Dart, **modern web applications** with React & Vite, and **robust backend APIs** using ASP.NET Core and Firebase. I love crafting real-time systems that feel alive — from live dashboards to collaborative tools.

Beyond code, I'm deeply interested in **AI tooling**, startup ecosystems, and building products that solve real problems at scale. Every project I build is a step toward understanding how modern software can be both powerful and beautifully simple.

- 🔭 Currently building: **NexVolt EV Charging Platform** & **ZmartPOS**
- 🌱 Learning: **LLM integrations**, **cloud-native architecture**, **system design**
- 💡 Passionate about: **mobile UX, real-time systems, AI-powered apps**
- 🤝 Open to: **internships, open source collabs, freelance projects**
- ⚡ Fun fact: I debug faster with lo-fi music 🎵
